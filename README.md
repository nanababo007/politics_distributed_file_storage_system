# politics_distributed_file_storage_system
정당의 모든 파일자료를 저정하는, 파일분산저장 php 시스템 프로젝트

https://blog.naver.com/dagabal/224281304402

민주구조의 모든 정당의 구조 업무수행이 위원회로 구성되고, 
위원회에서 모든 업무가 구성원 회의로 실무와 의사결정이 수행되고, 
투표로 의사결정이 수행되는, 정당의 위원회 회의 운영 시스템을 구축함.

이 시스템은, 위의 위원회로 운영되는 정당의 모든 업무에서 발생하는 파일의 저장을, 
부정부패 방지를 위해서 다수 조직이 운영하는 여러 서버들에 분산하여, 동일파일을 여러군데 서버에 
분산저장하고, 조회 시 랜덤으로 몇개의 서버에서 동일 파일을 불러와서 비교를 하여 조작여부를 수행하는 
기능의 프로그램을 구현하는 오픈 프로젝트.

프로그램 환경 : php, mariadb
